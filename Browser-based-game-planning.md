# Project planning

## Choice of Game
Wordle


## Pseudocode
1. Game display module will have 5 x 6 square blocks where letters will be entered into

2. On the bottom of the module, there should be all 24 letters of the alphabet, enter and delete button that the player can click to enter letters 

3. Player will enter 5 letter word to guess the given word
   
4. If the letter of the word they got is correct and is in the right position, it will turn green, if the letter is correct but in the wrong position it will turn yellow. If the word does not contain the input letter, it will stay grey
   
5. If the player guess the word right in 6 turns they win. if they cannot guess the word, they lose.
